# Titanic Survival Prediction

## 📌 Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning classification algorithms. It demonstrates the complete machine learning workflow, including data preprocessing, feature engineering, model training, and performance evaluation.

---

## 🎯 Objective

Build and compare multiple classification models to accurately predict passenger survival using the Titanic dataset.

---

## 📂 Dataset

- **Dataset:** Titanic Dataset
- **Problem Type:** Binary Classification
- **Target Variable:** `Survived`

---

## 🤖 Machine Learning Algorithms

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## ⚙️ Data Preprocessing

The following preprocessing techniques were applied:

- Handling missing values
- Removing unnecessary columns
- Encoding categorical features
- Feature scaling (where required)
- Train-Test Split

---

## 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/kevinjoelc/Titanic-Survival-Prediction.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Open Jupyter Notebook

```bash
jupyter notebook
```

Run any of the following notebooks:

- `titanic_logistic_regression.ipynb`
- `titanic_decision_tree.ipynb`
- `titanic_random_forest.ipynb`
